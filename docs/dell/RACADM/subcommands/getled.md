# getled

## Description

Displays the LED settings on a module: blinking, not blinking, or unknown (for empty slots).

To run this subcommand, you must have the Login User privilege.

## Synopsis

```
racadm getled
```

## Output/Example

- LED is blinking
- LED is not-blinking

```
racadm getled
```

```
LED State : Blinking
```

```
racadm getled
```

```
LED State : Not-Blinking
```
